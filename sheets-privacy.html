Privacy Policy — LinkSheet for Google Sheets

Last updated: June 1, 2026 · CropSettle LLC

LinkSheet for Google Sheets ("the add-on") is a Google Workspace add-on published by CropSettle LLC. This policy explains what the add-on accesses and what it does — and does not — do with your data.

What the add-on accesses

  • The active spreadsheet — to read cell values and write the hyperlinks you ask it to create (and remove them when you use Clear Hyperlinks). Scope: spreadsheets.currentonly (only the spreadsheet you have open, not your other files).
  • Your Google Drive (read-only, metadata only) — to read the files in the folder you choose and obtain each file's shareable link so it can build the hyperlinks. Scope: drive.metadata.readonly. You pick the folder via the Google Picker; the add-on reads only file names and links — never file contents, operates only on the folder you select, and never modifies, deletes, or creates anything in your Drive.

What the add-on does NOT do

  • It does not copy, store, or transmit your spreadsheet, your files, or their contents to CropSettle LLC or any third party.
  • It does not open, read, or use any files outside the folder you select for a run, and it never reads file contents — only names and links.
  • It does not sell or share any personal data.
  • It does not use your content for advertising or model training.

Processing & retention

All processing happens within Google's environment during the moment you run the add-on. The add-on keeps only minimal settings (e.g. your last-used options and, if you enter one, a license key) in Google's per-user/script property storage. It retains no copy of your spreadsheet or file contents after a run completes.

Undo & safety

The add-on does not create separate backup copies. Google Sheets' built-in version history (File → Version history) is the safety net — any run can be rolled back there.

Google API Services User Data Policy

LinkSheet for Google Sheets' use and transfer of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.
